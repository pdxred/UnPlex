## Cross-Phase Integration

32 of 34 cross-phase connections properly wired. 2 critical breaks found:

### CRITICAL: Auto-play Episode Metadata Not Wired

**From:** EpisodeScreen.brs / DetailScreen.brs (startPlayback)
**To:** VideoPlayer (parentRatingKey, grandparentRatingKey, episodeIndex, seasonIndex)
**Impact:** Auto-play countdown never triggers. VideoPlayer.brs line 953 checks `grandparentRatingKey` which is always empty because callers never set it.
**Affects:** PLAY-12, PLAY-13

**Fix:** Both `EpisodeScreen.startPlayback()` and `DetailScreen.startPlayback()` must set `m.player.parentRatingKey`, `m.player.grandparentRatingKey`, `m.player.episodeIndex`, and `m.player.seasonIndex` before starting playback.

### MEDIUM: watchStateChanged Not Observed

**From:** DetailScreen.brs (watchStateChanged field)
**To:** HomeScreen/MainScene (no observer)
**Impact:** Marking watched/unwatched on DetailScreen doesn't propagate to parent grid/hub rows until auto-refresh.
**Affects:** PLAY-04, PLAY-05
