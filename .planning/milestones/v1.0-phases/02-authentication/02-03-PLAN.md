Enhance PlexApiTask to detect 401 Unauthorized responses and signal that re-authentication is required.

**PlexApiTask.xml changes:**
- Add interface field: `responseCode` (integer, value=0) - HTTP response code from last request
- Update state comment to include: "authRequired" state for 401 responses

**PlexApiTask.brs changes:**

1. After getting response, capture response code:
   Find the line where `url.GetToString()` or `url.PostFromString()` is called.

   After the response is received, add:
   ```brightscript
   responseCode = url.GetResponseCode()
   m.top.responseCode = responseCode
   ```

2. Add 401 detection before JSON parsing:
   ```brightscript
   ' Check for 401 Unauthorized (token expired/invalid)
   if responseCode = 401
       LogError("401 Unauthorized - authentication required")
       ' Clear the stored token since it's invalid
       SetAuthToken("")
       ' Signal auth required via global
       m.global.addFields({ authRequired: true })
       m.global.authRequired = true
       m.top.error = "Authentication required"
       m.top.state = "authRequired"
       return
   end if
   ```

3. The 401 check should happen in both GET and POST code paths, right after receiving the response and capturing the response code.

4. Ensure this check happens BEFORE the empty response check (since 401 might have an empty body).

**Important:** The authRequired global field is observed by MainScene (added in Task 2). When set to true, MainScene will show the PIN screen.
  </action>
  <verify>
PlexApiTask.xml has responseCode field in interface. PlexApiTask.brs checks for responseCode = 401, clears token via SetAuthToken(""), sets m.global.authRequired = true, and sets state to "authRequired". Check appears in both GET and POST paths.
  </verify>
  <done>
PlexApiTask detects 401 responses, clears invalid token, signals global authRequired flag for MainScene to observe, enters "authRequired" state.
  </done>

  <action>
Add authentication flow integration to MainScene: check for stored credentials on launch, observe authRequired global, handle sign out.

**utils.brs additions:**
Add ClearAuthData function:
```brightscript
' Clear all stored authentication data (sign out)
sub ClearAuthData()
    sec = CreateObject("roRegistrySection", "PlexClassic")
    sec.Delete("authToken")
    sec.Delete("serverUri")
    sec.Delete("serverClientId")
    sec.Flush()
    LogEvent("Auth data cleared")
end sub
```

**MainScene.xml changes:**
- Add to interface:
  - `showSignOut` (boolean, value=false) - trigger to show sign out confirmation (for Settings screen to call)
- Ensure all required scripts are included (utils.brs, constants.brs, logger.brs)

**MainScene.brs changes:**

1. In init(), add global field setup and observer:
   ```brightscript
   ' Initialize global auth fields
   m.global.addFields({ authRequired: false })
   m.global.observeField("authRequired", "onAuthRequired")

   ' Check authentication status on launch
   checkAuthAndRoute()
   ```

2. Add checkAuthAndRoute() function:
   ```brightscript
   sub checkAuthAndRoute()
       token = GetAuthToken()
       serverUri = GetServerUri()

       if token = "" or serverUri = ""
           ' No credentials - show PIN screen
           LogEvent("No stored credentials, showing PIN screen")
           showPINScreen()
       else
           ' Have credentials - verify server is reachable
           LogEvent("Found stored credentials, attempting connection")
           ' For now, trust stored credentials and show home
           ' Server reachability will be validated on first API call
           ' If 401, onAuthRequired will trigger
           showHomeScreen()
       end if
   end sub
   ```

3. Add showPINScreen() function:
   ```brightscript
   sub showPINScreen()
       ' Clear any existing screens
       clearScreenStack()

       ' Create and show PIN screen
       pinScreen = CreateObject("roSGNode", "PINScreen")
       pinScreen.observeField("state", "onPINScreenState")
       pushScreen(pinScreen)
   end sub
   ```

4. Add onPINScreenState() callback:
   ```brightscript
   sub onPINScreenState(event as Object)
       state = event.getData()

       if state = "authenticated"
           ' PIN auth complete, get servers from PIN screen
           pinScreen = getCurrentScreen()
           servers = pinScreen.servers
           authToken = pinScreen.authToken

           if servers <> invalid and servers.count() > 1
               ' Multiple servers - show selection
               showServerListScreen(servers, authToken)
           else if servers <> invalid and servers.count() = 1
               ' Single server - auto-connect
               autoConnectToServer(servers[0], authToken)
           else
               ' No servers found (unusual)
               LogError("No servers found after authentication")
               ' Stay on PIN screen with error? Or show error screen?
           end if

       else if state = "cancelled"
           ' User cancelled - if we have credentials, go home; otherwise stay
           if GetAuthToken() <> "" and GetServerUri() <> ""
               popScreen()
               showHomeScreen()
           end if
       end if
   end sub
   ```

5. Add showServerListScreen() function:
   ```brightscript
   sub showServerListScreen(servers as Object, authToken as String)
       clearScreenStack()

       serverScreen = CreateObject("roSGNode", "ServerListScreen")
       serverScreen.servers = servers
       serverScreen.authToken = authToken
       serverScreen.observeField("state", "onServerListState")
       pushScreen(serverScreen)
   end sub
   ```

6. Add onServerListState() callback:
   ```brightscript
   sub onServerListState(event as Object)
       state = event.getData()

       if state = "connected"
           LogEvent("Server connected, showing home")
           clearScreenStack()
           showHomeScreen()
       else if state = "cancelled"
           ' Go back to PIN screen
           showPINScreen()
       end if
   end sub
   ```

7. Add autoConnectToServer() function:
   ```brightscript
   sub autoConnectToServer(server as Object, authToken as String)
       ' For single server, test connection then proceed
       m.connectionTask = CreateObject("roSGNode", "ServerConnectionTask")
       m.connectionTask.connections = server.connections
       m.connectionTask.authToken = authToken
       m.connectionTask.observeField("state", "onAutoConnectState")
       m.connectionTask.control = "run"

       ' Save server clientId while testing
       sec = CreateObject("roRegistrySection", "PlexClassic")
       sec.Write("serverClientId", server.clientId)
       sec.Flush()
   end sub
   ```

8. Add onAutoConnectState() callback:
   ```brightscript
   sub onAutoConnectState(event as Object)
       state = event.getData()

       if state = "connected"
           SetServerUri(m.connectionTask.successfulUri)
           LogEvent("Auto-connected to server")
           clearScreenStack()
           showHomeScreen()
       else if state = "error"
           LogError("Auto-connect failed: " + m.connectionTask.error)
           ' Show error - server unreachable
           ' Could show a "Can't reach server" screen with retry
           ' For now, show PIN screen again (user can retry auth)
           showPINScreen()
       end if
   end sub
   ```

9. Add onAuthRequired() observer callback:
   ```brightscript
   sub onAuthRequired(event as Object)
       required = event.getData()

       if required = true
           LogEvent("Auth required signal received, showing PIN screen")
           ' Reset the flag
           m.global.authRequired = false
           ' Show PIN screen
           showPINScreen()
       end if
   end sub
   ```

10. Add signOut() function (called from Settings screen via interface field):
    ```brightscript
    sub signOut()
        LogEvent("User signed out")
        ClearAuthData()
        showPINScreen()
    end sub
    ```

    Also observe the showSignOut field:
    ```brightscript
    ' In init()
    m.top.observeField("showSignOut", "onShowSignOut")

    sub onShowSignOut(event as Object)
        if event.getData() = true
            m.top.showSignOut = false  ' Reset
            signOut()
        end if
    end sub
    ```

11. Ensure screen stack helper functions exist (may already exist from scaffolding):
    - clearScreenStack() - removes all screens
    - pushScreen(screen) - adds screen to stack, sets focus
    - popScreen() - removes top screen
    - getCurrentScreen() - returns top of stack
