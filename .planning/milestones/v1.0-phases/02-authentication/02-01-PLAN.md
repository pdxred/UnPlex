Create a full-screen PIN display component per CONTEXT.md locked decisions.

**PINScreen.xml:**
- Extends Group, component name="PINScreen"
- Include scripts: utils.brs, constants.brs, logger.brs, PINScreen.brs
- Define interface fields:
  - `state` (string, value="idle") - tracks screen state: idle, loading, pinReady, authenticated, error
  - `authToken` (string) - populated when auth succeeds
  - `servers` (assocarray) - populated with server list after auth
- Layout children (FHD 1920x1080):
  - Background Rectangle (full screen, color from constants BG_PRIMARY)
  - Container Group centered vertically and horizontally containing:
    - Label for "plex.tv/link" URL (large font 48px, white text, positioned above PIN)
    - Label id="pinCodeLabel" for 4-digit PIN code (VERY large font 120px, Plex gold/ACCENT color, centered)
    - LoadingSpinner widget (below PIN, initially visible=false)
    - Label id="statusLabel" for status text (24px, gray, below spinner, e.g. "Waiting for authorization...")
    - Label id="errorLabel" for error messages (24px, red, visible=false)
  - Hint label at bottom: "Press Back to cancel"

**PINScreen.brs:**
- init():
  - Store references to UI elements (m.pinCodeLabel, m.statusLabel, m.errorLabel, m.spinner)
  - Create m.authTask = CreateObject("roSGNode", "PlexAuthTask")
  - Observe m.authTask.state via onAuthStateChange callback
  - Create m.pollTimer = CreateObject("roSGNode", "Timer") with duration=2 (2 second polling interval)
  - Observe m.pollTimer.fire via onPollTimer callback
  - Call startPinRequest() to initiate flow

- startPinRequest():
  - Set m.authTask.action = "requestPin"
  - Set m.authTask.control = "run"
  - Show spinner, set status text to "Requesting PIN..."

- onAuthStateChange(event):
  - Get state = event.getData()
  - Switch on state:
    - "pinReady":
      - Display PIN: m.pinCodeLabel.text = m.authTask.pinCode
      - Store m.currentPinId = m.authTask.pinId
      - Store m.pinExpiresAt from m.authTask.expiresAt (ISO 8601 string)
      - Set status text to "Waiting for authorization..."
      - Start poll timer: m.pollTimer.control = "start"
    - "waiting":
      - Check PIN expiration via checkPinExpiration()
      - If not expired, do nothing (timer will fire again)
    - "authenticated":
      - Stop poll timer
      - LogEvent("PIN authentication successful")
      - Set status text to "Success! Loading servers..."
      - Fetch servers: m.authTask.action = "fetchResources", m.authTask.control = "run"
    - "serversReady":
      - m.top.servers = m.authTask.servers
      - m.top.authToken = m.authTask.authToken
      - m.top.state = "authenticated"
    - "error":
      - Stop poll timer
      - Hide spinner
      - Show error label with m.authTask.error
      - LogError("PIN auth failed: " + m.authTask.error)
    - "refreshing":
      - Show status "Refreshing PIN..."

- checkPinExpiration():
  - Parse m.pinExpiresAt as roDateTime
  - Compare to current time
  - If < 30 seconds remaining: request new PIN automatically
  - LogEvent("PIN expiring, auto-refreshing")

- onPollTimer(event):
  - Set m.authTask.action = "checkPin"
  - Set m.authTask.pinId = m.currentPinId
  - Set m.authTask.control = "run"

- onKeyEvent(key, press):
  - If key = "back" and press = true:
    - Stop poll timer
    - m.authTask.control = "stop"
    - m.top.state = "cancelled"
    - return true
  - return false
  </action>

4. Add fetchResources() function:
   ```brightscript
   sub fetchResources()
       m.top.state = "loading"
       LogEvent("Fetching server resources")

       url = CreateObject("roUrlTransfer")
       url.SetCertificatesFile("common:/certs/ca-bundle.crt")
       url.InitClientCertificates()
       url.SetUrl("https://plex.tv/api/v2/resources?includeHttps=1&includeRelay=1")

       headers = GetPlexHeaders()
       headers["X-Plex-Token"] = m.top.authToken
       for each key in headers
           url.AddHeader(key, headers[key])
       end for

       response = url.GetToString()

       if response = ""
           m.top.error = "Failed to fetch servers: " + url.GetFailureReason()
           m.top.state = "error"
           return
       end if

       json = ParseJson(response)
       if json = invalid
           m.top.error = "Invalid server response"
           m.top.state = "error"
           return
       end if

       m.top.servers = parseServerList(json)
       LogEvent("Found " + m.top.servers.count().ToStr() + " server(s)")
       m.top.state = "serversReady"
   end sub
   ```

5. Add parseServerList() function:
   ```brightscript
   function parseServerList(json as Object) as Object
       servers = []
       devices = SafeGet(json, "MediaContainer", invalid)
       if devices = invalid then devices = json  ' Handle v2 API format

       deviceList = SafeGet(devices, "Device", [])
       if type(deviceList) <> "roArray" then deviceList = []

       for each device in deviceList
           provides = SafeGet(device, "provides", "")
           ' Only include servers (not players, etc.)
           if Instr(1, provides, "server") > 0
               serverInfo = {
                   name: SafeGet(device, "name", "Unknown Server")
                   clientId: SafeGet(device, "clientIdentifier", "")
                   version: SafeGet(device, "productVersion", "")
                   connections: parseConnections(SafeGet(device, "Connection", []))
               }
               servers.push(serverInfo)
           end if
       end for

       return servers
   end function
   ```

6. Add parseConnections() function:
   ```brightscript
   function parseConnections(connArray as Object) as Object
       result = { local: [], remote: [], relay: [] }

       if type(connArray) <> "roArray" then return result

       for each conn in connArray
           connInfo = {
               uri: SafeGet(conn, "uri", "")
               address: SafeGet(conn, "address", "")
               port: SafeGet(conn, "port", "32400")
               protocol: SafeGet(conn, "protocol", "http")
           }

           isLocal = SafeGet(conn, "local", "0") = "1" or SafeGet(conn, "local", 0) = 1
           isRelay = SafeGet(conn, "relay", "0") = "1" or SafeGet(conn, "relay", 0) = 1

           if isLocal
               result.local.push(connInfo)
           else if isRelay
               result.relay.push(connInfo)
           else
               result.remote.push(connInfo)
           end if
       end for

       return result
   end function
   ```
