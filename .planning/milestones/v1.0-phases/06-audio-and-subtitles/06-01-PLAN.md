<task type="auto">
  <name>Task 1: Create TrackSelectionPanel component with slide animation and track display</name>
  <files>SimPlex/components/widgets/TrackSelectionPanel.xml, SimPlex/components/widgets/TrackSelectionPanel.brs</files>
  <action>
Create a SceneGraph component that slides in from the right edge of the screen.

**TrackSelectionPanel.xml:**
- Extends Group
- Interface fields:
  - `audioStreams` (array) — array of audio stream AAs from Plex API
  - `subtitleStreams` (array) — array of subtitle stream AAs from Plex API
  - `selectedAudioId` (integer) — currently selected audio stream ID
  - `selectedSubtitleId` (integer) — currently selected subtitle stream ID (0 = off)
  - `visible` (boolean) — controls panel visibility
  - `trackChanged` (assocarray, alwaysNotify=true) — fires when user selects a track: { type: "audio"|"subtitle", streamId: int, codec: string, isBitmap: bool }
- Children:
  - Background Rectangle: width ~450, height 1080, color semi-transparent dark (e.g. "0x000000CC"), translation starts off-screen at [1920, 0]
  - LayoutGroup (vertical) inside background:
    - Audio section: Label header "Audio" (white, bold, ~28pt), then LabelList for audio tracks
    - Divider: Rectangle separator (1px height, SEPARATOR color)
    - Subtitle section: Label header "Subtitles" (white, bold, ~28pt), then LabelList for subtitle tracks
  - Animation node with FloatFieldInterpolator for slide-in/slide-out (translateX from 1920 to ~1470)
- Include script refs: TrackSelectionPanel.brs, pkg:/source/utils.brs, pkg:/source/constants.brs

**TrackSelectionPanel.brs:**
- `init()`: Find child nodes, set up animation references, create "Off" entry for subtitles
- `onAudioStreamsChange()`: Populate audio LabelList with ContentNode children. Each item's title = stream.displayTitle (e.g. "English (AAC 5.1)"). If displayTitle is empty, construct from language + codec + channels. Store stream ID on each ContentNode via addFields.
- `onSubtitleStreamsChange()`: Same for subtitles. First item is always "Off" with streamId=0. Append all subtitle streams. Label = stream.displayTitle (e.g. "English (SRT)" or "English (PGS)"). Store stream ID, codec, and isBitmap flag on each ContentNode.
- `onVisibleChange()`: When visible becomes true, run slide-in animation, set focus on audio LabelList, highlight current selection. When false, run slide-out animation.
- `highlightSelected()`: For each list, find the item matching selectedAudioId/selectedSubtitleId. Set that item's color to constants.ACCENT. Prepend a checkmark character (unicode ✓) to the selected item's title. Ensure previously selected items have their accent removed.
- `onAudioItemSelected()`: Read selected ContentNode's streamId. Fire trackChanged with { type: "audio", streamId: id, codec: codec, isBitmap: false }.
- `onSubtitleItemSelected()`: Read selected ContentNode's streamId, codec, isBitmap. Fire trackChanged with { type: "subtitle", streamId: id, codec: codec, isBitmap: isBitmap }.
- `onKeyEvent()`: Back or Options key closes panel (set visible=false). Up/Down at boundaries: bottom of audio list moves focus to subtitle list, top of subtitle list moves focus back to audio list. Return true to consume keys and prevent VideoPlayer from handling them.

Use m.global.constants for all colors (ACCENT, TEXT_PRIMARY, TEXT_SECONDARY, SEPARATOR, BG_PRIMARY).
Panel width: Claude's discretion (~400-450px).
Animation: 0.2-0.3 second ease-in-out for slide.

<task type="auto">
  <name>Task 2: Wire TrackSelectionPanel into VideoPlayer with stream metadata caching and track switching</name>
  <files>SimPlex/components/widgets/VideoPlayer.xml, SimPlex/components/widgets/VideoPlayer.brs</files>
  <action>
Extend VideoPlayer to parse stream metadata, show the track panel on Options key, and handle audio/SRT subtitle switching.

**VideoPlayer.xml changes:**
- Add TrackSelectionPanel as a child (initially not visible)
- Add interface fields:
  - `audioStreams` (array) — parsed audio stream data for external access
  - `subtitleStreams` (array) — parsed subtitle stream data for external access

**VideoPlayer.brs changes:**

1. **Stream metadata parsing in processMediaInfo():**
   After extracting `part` (which already happens), also extract `part.Stream` array. Call new `parseStreams(streams)` sub that builds two arrays:
   - `m.audioStreams`: array of AAs with { id, displayTitle, language, codec, channels, selected }
   - `m.subtitleStreams`: array of AAs with { id, displayTitle, language, codec, forced, selected, isBitmap }
   - `isBitmap` = true if codec is "pgs" or "vobsub" (case-insensitive)
   Store the `part.id` as `m.partId` for use in Plan 02's track persistence.
   Set m.top.audioStreams and m.top.subtitleStreams for the panel.
   Determine initially selected audio/subtitle IDs from streams with `selected: true`.

2. **Set up SRT sidecar subtitles before playback:**
   In processMediaInfo(), after building the ContentNode, check if any subtitle stream is selected AND is a text type (not bitmap). If so, build sidecar URL: `{serverUri}/library/streams/{streamId}?X-Plex-Token={token}` and set it on ContentNode's SubtitleTracks field as an array of { TrackName: "sidecar_{id}", Language: language, Url: sidecarUrl }.

3. **Panel creation and management:**
   In init(), find or create reference to the TrackSelectionPanel child node. Observe its `trackChanged` field.
   Add `m.trackPanel` reference. After streams are parsed (end of processMediaInfo), set panel's audioStreams, subtitleStreams, selectedAudioId, and selectedSubtitleId.

4. **Options key handling in onKeyEvent():**
   Add "options" key handler. When pressed:
   - If panel is visible, hide it (panel.visible = false), resume playback (m.video.control = "resume"), restore focus to m.video
   - If panel is not visible, pause playback (m.video.control = "pause"), set panel.visible = true

5. **Track change handler - onTrackChanged():**
   Read the trackChanged AA.

   If type = "audio":
   - Update m.selectedAudioId
   - For direct play: The simplest approach is to use Roku Video node's `audioTrack` field. Match the stream by index in m.audioStreams to find the 0-based track index. Set m.video.audioTrack = index.
   - Update panel's selectedAudioId

   If type = "subtitle" AND isBitmap = false (SRT/text):
   - If streamId = 0 (Off): call m.video.disableSubtitle() or set enableSubtitle to false
   - Otherwise: build sidecar URL, update ContentNode's SubtitleTracks, call m.video.enableSubtitle("sidecar_{streamId}")
   - Update m.selectedSubtitleId and panel's selectedSubtitleId

   If type = "subtitle" AND isBitmap = true (PGS):
   - Store pending PGS request: m.pendingPgsStreamId = streamId (Plan 02 will handle the actual transcode pivot)
   - For now, set a field `m.top.pgsRequested` (assocarray) with { streamId: id, position: m.currentPosition } so Plan 02 can observe and handle it

6. **Panel close behavior:**
   When panel becomes not visible (observe panel.visible), resume playback if it was paused for the panel (use a flag m.pausedForPanel to distinguish from user-initiated pause).

Do NOT modify the existing buildTranscodeUrl() function — Plan 02 will add the subtitle-specific variant.
Do NOT add PUT support to PlexApiTask — Plan 02 handles track persistence.
  </action>
