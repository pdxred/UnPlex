**VideoPlayer.brs changes:**

1. **In init():** Add auto-play node references and state:
   - `m.autoPlayOverlay = m.top.findNode("autoPlayOverlay")`
   - `m.autoPlayEpisodeLabel = m.top.findNode("autoPlayEpisodeLabel")`
   - `m.autoPlayCountdown = m.top.findNode("autoPlayCountdown")`
   - `m.autoPlayFocusBorder = m.top.findNode("autoPlayFocusBorder")`
   - `m.autoPlayFadeIn = m.top.findNode("autoPlayFadeIn")`
   - `m.autoPlayFadeOut = m.top.findNode("autoPlayFadeOut")`
   - `m.autoPlayFadeOut.observeField("state", "onAutoPlayFadeOutComplete")`
   - `m.nextEpisodeInfo = invalid` (AA with ratingKey, mediaKey, title, seasonEp)
   - `m.countdownSeconds = 10`
   - `m.countdownActive = false`
   - `m.autoPlayOverlayVisible = false`
   - `m.autoPlayFocused = false`
   - Create countdown timer:
     ```brightscript
     m.countdownTimer = CreateObject("roSGNode", "Timer")
     m.countdownTimer.duration = 1
     m.countdownTimer.repeat = true
     m.countdownTimer.observeField("fire", "onCountdownTick")
     ```

2. **Modify checkMarkers():** When position enters credits range AND `m.grandparentRatingKey` is set (TV episode) AND next episode info is available:
   - Instead of showing skip button, show auto-play countdown overlay
   - If next episode hasn't been fetched yet, fetch it first, then show on callback
   - If no next episode exists, fall back to showing the skip button (Phase 7 behavior)

   The logic: in checkMarkers(), when `inCredits` is true:
   - If `m.top.grandparentRatingKey <> ""` (TV episode):
     - If `m.nextEpisodeInfo <> invalid`: call `showAutoPlayOverlay()` instead of `showSkipButton("credits")`
     - If `m.nextEpisodeInfo = invalid` and not already fetching: call `fetchNextEpisode()`
     - If next episode confirmed to not exist (`m.noNextEpisode = true`): call `showSkipButton("credits")` as usual
   - If not a TV episode: call `showSkipButton("credits")` as usual

   Also add a fallback check: if NO credits marker exists, check if position >= 90% of duration. If so, trigger the same auto-play logic.

3. **New function fetchNextEpisode():** Fetch episodes in current season to find next:
   ```brightscript
   sub fetchNextEpisode()
       if m.top.parentRatingKey = "" then return
       m.fetchingNextEpisode = true
       task = CreateObject("roSGNode", "PlexApiTask")
       task.endpoint = "/library/metadata/" + m.top.parentRatingKey + "/children"
       task.observeField("status", "onNextEpisodeLoaded")
       task.control = "run"
       m.nextEpisodeTask = task
   end sub
   ```

4. **New function onNextEpisodeLoaded(event):**
   - Parse response.MediaContainer.Metadata array (list of episodes)
   - Find current episode by m.top.episodeIndex
   - If next episode exists (index + 1): store in m.nextEpisodeInfo = { ratingKey, mediaKey, title, seasonEp }
     - `mediaKey` = `/library/metadata/{ratingKey}`
     - `title` = episode title
     - `seasonEp` = "S{parentIndex} E{index}" formatted string
   - If next episode NOT found (last in season): try fetching next season
     - Fetch `/library/metadata/{grandparentRatingKey}/children` to get seasons
     - Find season with index = currentSeasonIndex + 1
     - If found, fetch that season's `/children` and get first episode
     - This requires chaining API calls. For simplicity, set `m.noNextEpisode = true` if current season is exhausted (cross-season auto-play as stretch)
   - Set `m.fetchingNextEpisode = false`
   - If in credits range and next episode found, trigger `showAutoPlayOverlay()`

5. **New function showAutoPlayOverlay():**
   - If already showing or `m.nextEpisodeInfo = invalid`, return
   - Hide skip button if visible (countdown replaces it)
   - Set episode label: `m.nextEpisodeInfo.seasonEp + " - " + m.nextEpisodeInfo.title`
   - Set countdown text: "Starting in 10..."
   - Set `m.countdownSeconds = 10`
   - Show overlay: `m.autoPlayOverlay.visible = true`, run fadeIn
   - Set `m.autoPlayOverlayVisible = true`
   - Start countdown timer: `m.countdownTimer.control = "start"`
   - Set `m.countdownActive = true`
   - Focus management: take focus if track panel not open
   - Show focus border with accent color

6. **New function hideAutoPlayOverlay():**
   - Stop countdown timer
   - Set `m.countdownActive = false`
   - Run fadeOut animation
   - Set `m.autoPlayOverlayVisible = false`
   - Return focus to video if had focus

7. **New function onAutoPlayFadeOutComplete(event):** Hide overlay on animation complete.

8. **New function onCountdownTick(event):**
   - Decrement `m.countdownSeconds`
   - Update countdown label: "Starting in {N}..."
   - If `m.countdownSeconds <= 0`: call `startNextEpisode()`

9. **New function startNextEpisode():**
   - Stop countdown timer
   - Scrobble current episode (call scrobble())
   - Stop current playback (but don't trigger playbackComplete)
   - Set new playback fields:
     - `m.top.ratingKey = m.nextEpisodeInfo.ratingKey`
     - `m.top.mediaKey = m.nextEpisodeInfo.mediaKey`
     - `m.top.itemTitle = m.nextEpisodeInfo.title`
     - `m.top.startOffset = 0` (or resume position if available)
   - Signal that next episode is starting: `m.top.nextEpisodeStarted = true`
   - Reset auto-play state (m.nextEpisodeInfo = invalid, m.introMarker = invalid, m.creditsMarker = invalid)
   - Hide auto-play overlay
   - Call `loadMedia()` to start the next episode

10. **Extend onKeyEvent():** Add auto-play overlay handling BEFORE skip button handling:
    - If `m.autoPlayFocused` and key = "OK": call `startNextEpisode()`, return true (immediate play)
    - If `m.autoPlayFocused` and key = "back": call `cancelAutoPlay()`, return true
    - `cancelAutoPlay()`: stop timer, hide overlay, return focus to video

11. **Extend stopPlayback():** Reset auto-play state — stop countdown timer, hide overlay, clear nextEpisodeInfo.

12. **Handle pause during countdown:** In the existing play/pause handler, if `m.countdownActive`:
    - On pause: stop countdown timer (pause it)
    - On resume: restart countdown timer

**Important implementation notes:**
- The 90% duration fallback: `if m.creditsMarker = invalid and pos >= m.duration * 0.9 and m.duration > 0`
- Auto-play only for TV episodes: check `m.top.grandparentRatingKey <> ""`
- Cross-season next episode is a stretch — for now, set `m.noNextEpisode = true` when last episode of season. This can be enhanced later.
- The `nextEpisodeStarted` interface field allows MainScene/calling screens to update their state (e.g., update episode list focus)
