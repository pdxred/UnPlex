**FilterBottomSheet.xml:**
- extends="Group"
- Interface fields:
  - visible (boolean, onChange="onVisibleChange")
  - sectionId (string, onChange="onSectionIdChange") -- triggers genre/year fetch
  - sectionType (string) -- "1" for movies, "2" for shows
  - filterState (assocarray, alwaysNotify=true) -- output: current filter state AA
  - sheetDismissed (boolean, alwaysNotify=true) -- signals parent to restore focus
  - genreDisplayNames (type assocarray) -- set after genres load so HomeScreen can pass to FilterBar

- Scripts: link utils.brs, constants.brs, FilterBottomSheet.brs

- Children structure (all inside a Group id="sheetGroup" translation="[280, 1080]" -- starts offscreen below, positioned at sidebar width offset):
  - Rectangle id="sheetBg" width=1640 height=500 color=0x1E1E3AFF (dark background, slightly lighter than main bg)
  - Rectangle id="handleBar" translation=[770, 8] width=100 height=4 color=0x505070FF (visual grab handle)

  - Scrollable content area (Group id="controlsGroup" translation=[40, 40]):
    - Label "Sort" header: font SmallBoldSystemFont, color 0xE5A00DFF (accent), translation [0, 0]
    - RadioButtonList id="sortList" translation=[0, 35] itemSize=[350, 48] -- 8 items for sort options
    - Label "Unwatched" header: translation [400, 0], font SmallBoldSystemFont, color 0xE5A00DFF
    - LabelList id="unwatchedList" translation=[400, 35] itemSize=[350, 48] -- 2 items: "All Items", "Unwatched Only"
    - Label "Genre" header: translation [400, 140], font SmallBoldSystemFont, color 0xE5A00DFF
    - CheckList id="genreList" translation=[400, 175] itemSize=[350, 48] numRows=6 -- scrollable
    - Label "Year" header: translation=[800, 0], font SmallBoldSystemFont, color 0xE5A00DFF
    - LabelList id="yearList" translation=[800, 35] itemSize=[350, 48] numRows=8 -- decades + individual years, scrollable
    - ButtonGroup id="clearAllButton" translation=[1200, 0] buttons=["Clear All"]

  Note: Per user decision, sort and filters are mixed together in the sheet (not separate sections). The layout uses columns side by side rather than separate tabbed sections. Sort on the left, unwatched + genre in the middle, year + clear on the right.

**FilterBottomSheet.brs:**

- init():
  - Find all nodes: sheetGroup, sortList, unwatchedList, genreList, yearList, clearAllButton, slideUpAnim, slideDownAnim
  - m.isOpen = false
  - m.genreKeys = [] (maps CheckList index to API genre key)
  - m.genreNames = {} (maps genre key to display name for FilterBar)
  - m.yearValues = [] (maps LabelList index to year value or decade range)
  - m.currentFilterState = { sort: "titleSort:asc", genre: "", year: "", unwatched: "" }
  - Populate sort list with 8 options: Title A-Z, Title Z-A, Date Added (Newest), Date Added (Oldest), Year (Newest), Year (Oldest), Rating (Highest), Rating (Lowest)
  - Populate unwatched list with 2 options: "All Items", "Unwatched Only"
  - Observe sortList.checkedItem -> onSortChanged
  - Observe unwatchedList.itemSelected -> onUnwatchedChanged (Note: LabelList uses itemSelected, not checkedItem)
  - Observe genreList.checkedState -> onGenreChanged
  - Observe clearAllButton.buttonSelected -> onClearAll

- populateSortList(): Create ContentNode tree with 8 sort option titles. Set m.sortValues = ["titleSort:asc", "titleSort:desc", "addedAt:desc", "addedAt:asc", "year:desc", "year:asc", "rating:desc", "rating:asc"]. Set sortList.checkedItem = 0.

- onSectionIdChange(event): Fetch genre and year lists for the new section. Reset genre/year selections.

- loadGenres(): Create PlexApiTask, endpoint = "/library/sections/" + m.top.sectionId + "/genre", params = { "type": m.top.sectionType }. Observe status -> onGenresLoaded.

- onGenresLoaded(event): Parse response. Build ContentNode tree for genreList from response Directory array. Each item has title field. Store m.genreKeys[i] = item.key (the API filter key). Store m.genreNames[item.key] = item.title. Reset genreList.checkedState to all false. Set m.top.genreDisplayNames = m.genreNames.

- loadYears(): Create PlexApiTask, endpoint = "/library/sections/" + m.top.sectionId + "/year", params = { "type": m.top.sectionType }. Observe status -> onYearsLoaded.

- onYearsLoaded(event): Parse response. Group years into decades (2020s, 2010s, etc.) plus individual years within the most recent decade. Build ContentNode tree for yearList. Store m.yearValues[i] = the year value string (single year) or comma-separated range (decade like "2020,2021,2022,...,2029"). Add "All Years" as first item with empty value.

- onSortChanged(event): index = m.sortList.checkedItem. m.currentFilterState.sort = m.sortValues[index]. Emit filter state.

- onUnwatchedChanged(event): index = event.getData(). If index = 0 then m.currentFilterState.unwatched = "" else m.currentFilterState.unwatched = "1". Emit filter state.

- onGenreChanged(event): Read m.genreList.checkedState array. Build comma-separated string of selected genre keys from m.genreKeys. Set m.currentFilterState.genre = result. Emit filter state.

- onYearSelected(event): index = event.getData(). m.currentFilterState.year = m.yearValues[index]. Emit filter state.

- onClearAll(event):
  - Reset m.currentFilterState to defaults: { sort: "titleSort:asc", genre: "", year: "", unwatched: "" }
  - Reset sortList.checkedItem = 0
  - Reset unwatchedList (set focus to item 0)
  - Reset genreList.checkedState to all false
  - Reset yearList (set focus to item 0 "All Years")
  - Emit filter state

- emitFilterState(): Set m.top.filterState = m.currentFilterState. This is observed by HomeScreen which passes it to FilterBar.filterState.

- openSheet():
  - m.sheetGroup.visible = true
  - m.slideUpAnim.control = "start"
  - m.isOpen = true
  - m.sortList.setFocus(true) (default focus to sort list)

- closeSheet():
  - m.slideDownAnim.observeField("state", "onSlideDownComplete")
  - m.slideDownAnim.control = "start"

- onSlideDownComplete(event):
  - If event.getData() = "stopped":
    - m.sheetGroup.visible = false
    - m.slideDownAnim.unobserveField("state")
    - m.isOpen = false
    - m.top.sheetDismissed = true

- onVisibleChange(event): If m.top.visible = true then openSheet() else closeSheet()

- onKeyEvent(key, press):
  - If not press or not m.isOpen: return false
  - If key = "back": closeSheet(), return true
  - Focus navigation between columns:
    - If key = "right": Move focus from sortList -> unwatchedList -> genreList -> yearList -> clearAllButton
    - If key = "left": Reverse direction
    - Consume all left/right/up/down to trap focus within sheet (per research pitfall #3)
  - Return true for all keys when sheet is open (focus trapping) EXCEPT do not consume "back"
  - Actually: return false for up/down within lists (let the list handle scrolling), but consume left/right for column navigation. Back always closes. OK is handled by list components.
  - Refined approach: Only consume "left" and "right" for cross-column navigation. Let up/down/OK pass through to the focused list. Consume "back" to close.
  </action>

       - m.filterBottomSheet.visible = true
       - m.isSheetOpen = true
       - return true
   - When m.viewMode = "hubGrid": keep existing hub row options menu behavior unchanged
   - Remove the openFilterSheet interface field added in Plan 01 if it was added (the sheet is now directly in HomeScreen, not triggered via MainScene)
