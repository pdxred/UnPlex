Add a RowList for hub rows to the HomeScreen and fetch hub data from the Plex /hubs API.

**HomeScreen.xml changes:**
Inside the contentArea Group, add a RowList BEFORE the FilterBar:
```xml
<RowList
    id="hubRowList"
    translation="[0, 0]"
    itemSize="[1620, 430]"
    numRows="0"
    rowItemSize="[[260, 390]]"
    rowItemSpacing="[[20, 0]]"
    itemSpacing="[0, 10]"
    showRowLabel="[true]"
    rowLabelOffset="[[0, 0]]"
    drawFocusFeedback="true"
    focusBitmapBlendColor="0xE5A00DFF"
    vertFocusAnimationStyle="floatingFocus"
    rowFocusAnimationStyle="floatingFocus"
    itemComponentName="PosterGridItem"
    visible="false"
/>
```

Note: numRows starts at 0 (will be set dynamically), itemSize width=1620 (1920 - 280 sidebar - 20 padding), rowItemSize 260x390 matches PosterGridItem (240 poster + 20 spacing, 360 poster + 30 title area). The RowList reuses PosterGridItem as itemComponentName.

**HomeScreen.brs changes:**

1. In init(), add:
   - `m.hubRowList = m.top.findNode("hubRowList")`
   - `m.hubRowCount = 0`
   - `m.isLoadingHubs = false`
   - `m.hubRowMap = {}`  (maps row index to hub type string)
   - `m.focusArea = "sidebar"` (tracks current focus zone: "sidebar", "hubs", "grid")
   - Call `loadHubs()` at end of init

2. Add `loadHubs()` sub:
   - Guard: `if m.isLoadingHubs then return`
   - Set `m.isLoadingHubs = true`
   - Create a new PlexApiTask with endpoint = "/hubs" and params = {}
   - Observe "status" field with callback "onHubsLoaded"
   - Set control = "run"
   - Store as `m.hubsTask = task`

3. Add `onHubsLoaded(event as Object)` sub:
   - Set `m.isLoadingHubs = false`
   - Guard: if event.getData() <> "completed" then return
   - Guard: if m.hubsTask.response = invalid or m.hubsTask.response.MediaContainer = invalid then return
   - Get hubs array: `hubs = m.hubsTask.response.MediaContainer.Hub`
   - If hubs = invalid then return
   - Initialize: m.continueWatching = invalid, m.onDeck = invalid, m.recentlyAdded = invalid
   - Loop through hubs, using LCase(hub.hubIdentifier) for case-insensitive matching:
     - If contains "continue" -> m.continueWatching = hub
     - Else if contains "ondeck" -> m.onDeck = hub
     - Else if contains "recent" -> m.recentlyAdded = hub (take first match only)
   - Use Instr(1, id, "continue") > 0 pattern for partial matching
   - Call buildHubRowContent()

4. Add `buildHubRowContent()` sub:
   - Create rootContent = CreateObject("roSGNode", "ContentNode")
   - Reset m.hubRowMap = {}
   - Track rowIndex = 0
   - Get constants: c = m.global.constants
   - Per locked decision, add rows in order: Continue Watching, On Deck, Recently Added
   - For each hub (if not invalid), check Metadata array exists and count() > 0
   - If has data: call addHubRow(rootContent, title, metadata, c), store hubRowMap[rowIndex.ToStr()] = hubType string, increment rowIndex
   - CRITICAL ordering: set m.hubRowList.numRows = rowIndex BEFORE setting content
   - Set m.hubRowList.content = rootContent
   - Set m.hubRowList.visible = (rowIndex > 0)
   - Store m.hubRowCount = rowIndex
   - Call repositionContentBelowHubs(rowIndex)

5. Add `addHubRow(rootContent as Object, title as String, metadata as Object, c as Object)` sub:
   - Create rowNode = rootContent.createChild("ContentNode")
   - Set rowNode.title = title (RowList uses TITLE for row labels)
   - Loop through metadata items, creating child ContentNodes with:
     - title, ratingKey (as string, handle both string and integer types), itemType (from item.type)
     - viewOffset (default 0, set from item.viewOffset if present)
     - duration (default 0, set from item.duration if present; fallback to item.Media[0].duration if available)
     - HDPosterUrl built via BuildPosterUrl(item.thumb, c.POSTER_WIDTH, c.POSTER_HEIGHT) if thumb exists
   - Use addFields to set custom fields on ContentNode

6. Add `repositionContentBelowHubs(hubRowCount as Integer)` sub:
   - If hubRowCount = 0: position filterBar at [20, 20], posterGrid at [20, 100] (existing defaults)
   - Else: calculate hubHeight = hubRowCount * 470 (430 item height + 10 spacing + 30 label area), position filterBar at [20, hubHeight + 20], posterGrid at [20, hubHeight + 100]

7. Update cleanup() sub to also:
   - Stop and unobserve m.hubsTask if not invalid
   - Unobserve hubRowList fields

8. Replace the simple m.focusOnSidebar boolean with the m.focusArea string. Update onFocusChange to delegate focus based on m.focusArea ("sidebar" -> sidebar, "hubs" -> hubRowList, "grid" -> posterGrid). Update onKeyEvent left/right handling to use m.focusArea instead of m.focusOnSidebar.
  </action>
