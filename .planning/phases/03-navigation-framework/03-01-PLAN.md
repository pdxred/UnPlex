Note: Sidebar component already exists and displays library list (libraryList, hubList, bottomList LabelLists) from Phase 2. This plan adds cleanup discipline, not Sidebar functionality.

Output: Enhanced MainScene.brs with cleanupScreen() helper, proper unobserveField calls, and validity-checked focus restoration.
</objective>

<task type="auto">
  <name>Task 1: Add cleanupScreen helper and enhance popScreen</name>
  <files>PlexClassic/components/MainScene.brs</files>
  <action>
Add a cleanupScreen() sub that handles observer cleanup before screen removal:

```brightscript
sub cleanupScreen(screen as Object)
    ' Unobserve all standard screen fields
    screen.unobserveField("itemSelected")
    screen.unobserveField("navigateBack")
    screen.unobserveField("state")

    ' Optional: Call screen's own cleanup if it exists
    if screen.hasField("cleanup")
        screen.callFunc("cleanup")
    end if
end sub
```

Update popScreen() to call cleanupScreen before removeChild:
- After `currentScreen = m.screenStack.pop()`
- Add `cleanupScreen(currentScreen)`
- Then `m.screenContainer.removeChild(currentScreen)`

Update focus restoration in popScreen() to check validity:
- Change `if savedFocus <> invalid` to `if savedFocus <> invalid and savedFocus.isValid()`
- This prevents setFocus on a node that was removed from scene graph

Update clearScreenStack() to call cleanupScreen for each screen:
- In the while loop, call `cleanupScreen(screen)` before `m.screenContainer.removeChild(screen)`

    ' Stop running task
    if m.apiTask <> invalid
        m.apiTask.control = "stop"
        m.apiTask.unobserveField("state")
    end if

    m.sidebar.unobserveField("selectedLibrary")
    m.sidebar.unobserveField("specialAction")
    m.posterGrid.unobserveField("itemSelected")
    m.posterGrid.unobserveField("loadMore")
