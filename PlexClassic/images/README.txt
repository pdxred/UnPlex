PlexClassic Image Assets
========================

Required images for the app to function properly:

1. icon_focus_fhd.png (540x405) - App icon shown in Roku home screen when focused
2. icon_side_fhd.png (108x108) - App icon shown in side panel
3. splash_fhd.jpg (1920x1080) - Splash screen shown during app launch
4. placeholder_poster.png (240x360) - Placeholder for missing poster images
5. spinner.png (80x80) - Loading spinner animation frame

Image Guidelines:
- All images should be in PNG format (except splash which can be JPG)
- Use Plex gold (#E5A00D) as accent color
- Background should match app theme (#1A1A2E)
- Keep file sizes optimized for fast loading

To create these images:
1. Use any image editor (Photoshop, GIMP, Figma, etc.)
2. Create images at the specified dimensions
3. Save with the exact filenames listed above
4. Place in this /images directory
